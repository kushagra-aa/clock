## Run Locally

Clone the project

```bash
  git clone https://github.com/kushagra-aa/clock.git
```

Go to the project directory

```bash
  cd clock
```

Install dependencies

```bash
  npm i
```

Development Build

Runs the app in the development mode.

```bash
  npm run tauri dev
```

The App Window will automatically appear
